Приложение использует сервер баз данных HSQLDB в in-process режиме
Является полностью рабочим с автоматическим заполнением базы данных тестовыми значениями
Конфигурация для запуска с рабочим дебагом: spring-boot:run -Dspring-boot.run.fork=false -f pom.xml

Открывается по веб адресу http://localhost:8080 в браузере.
